Wallpaper randomizer for KDE Plasma 5.24+ and Windows 10/11. 

As of now, it does not support any other desktop environment/operating systems.
This has only been tested on Arch Linux running KDE Plasma 6.6, and Windows 11 25h2, though it should run fine on Windows 10 and other Linux distributions.

Simply clone this repository, run the python file for your OS, and follow setup instructions. Afterwards, you may relaunch the program to quickly randomize your wallpaper whenever you want.
